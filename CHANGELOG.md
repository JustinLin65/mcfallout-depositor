# Changelog

本專案的所有顯著變更將記錄在此檔案中。
格式參考自 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/)。

## [1.1.0] - 2026-04-17

### Changed

- **存入邏輯優化**：改用伺服器內建的 `/moneysave` 指令取代原本需要開啟銀行 GUI 並點擊按鈕的流程。
- **移除冗餘動作**：刪除了對 `COORD_C` 的點擊動作以及按下 `e` 鍵關閉介面的步驟。
- **效率提升**：減少了與伺服器 GUI 的互動等待時間，整體循環更流暢。

## [1.0.0] - 2026-04-17

### Added

- **專案初始版本發佈**。
- **核心自動化邏輯**：實現右鍵開盒、Shift 快速移動、指令銀行存入等功能。
- **安全監控系統**：導入 `keyboard` 監聽與 `Esc` 鍵中斷機制。
- **穩定性優化**：細化點擊動作（mouseDown/mouseUp）以防丟包。
