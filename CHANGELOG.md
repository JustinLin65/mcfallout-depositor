# Changelog

本專案的所有顯著變更將記錄在此檔案中。
格式參考自 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/)。

## [1.0.0] - 2026-04-17

### Added

- **專案初始版本發佈**
- **核心自動化邏輯**：實現右鍵開盒、Shift 快速移動、指令銀行存入等功能。
- **安全監控系統**：
    - 導入 `keyboard` 監聽，支援 `Esc` 鍵即時中斷。
    - 實作 `safe_sleep` 函數，確保等待期間仍可捕捉停止指令。
- **穩定性優化**：針對滑鼠點擊動作加入 `mouseDown` 與 `mouseUp` 細節控制，減少丟包機率。
- **日誌輸出**：詳細的步驟說明。
