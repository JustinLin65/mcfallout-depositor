# Changelog

本專案的所有顯著變更將記錄在此檔案中。
格式參考自 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/)。

## [2.0.0] - 2026-04-17

### Added

- **智慧型保險機制**：新增基於 `pyautogui.pixel` 的螢幕偵測功能。若在 Shift 點擊後發現顏色無變化，代表有存款失敗，程式會自動按下 `e` 鍵跳出背包介面，有效防止網路波動導致的死循環。
- **影像處理依賴**：在 `requirements.txt` 中加入對像素偵測的支援說明。

### Changed

- **啟動流程優化**：將初始的 5 秒準備時間移出 `while True` 循環，避免每次循環都產生不必要的等待，大幅提升作業效率。

## [1.1.0] - 2026-04-17

### Changed

- **存入邏輯優化**：改用伺服器內建的 `/moneysave` 指令取代原本的 GUI 銀行存入流程。
- **移除冗餘動作**：刪除了對銀行按鈕的點擊動作與手動介面關閉。

## [1.0.0] - 2026-04-17

### Added

- **專案初始版本發佈**。
- **核心自動化邏輯**：實現右鍵開盒、Shift 點擊與中斷機制。
