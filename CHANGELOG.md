# Changelog

本專案的所有顯著變更將記錄在此檔案中。
格式參考自 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/)。

## [2.1.1] - 2026-04-17

### Changed

- **操作邏輯優化**：移除 `COORD_A` (固定開盒座標)。現在腳本會根據 Minecraft 機制直接在當前準星位置右鍵點擊開盒。
- **變數重命名**：將 `COORD_B` 統一改名為 `COORD_BATCH_MOVE`，明確標註其為點擊模組「批量移動」按鈕的座標。

## [2.1.0] - 2026-04-17

### Added

- **高速指令輸入**：引入 `pyperclip` 支援。現在 `/moneysave` 改用 `Ctrl+V` 貼上，不再逐字鍵入，顯著提升單次循環效率。
- **依賴更新**：在 `requirements.txt` 中加入 `pyperclip`。

### Changed

- **延遲優化**：調整 `pyautogui.PAUSE` 與指令間的等待時間，配合貼上機制達到更流暢的作業。

## [2.0.0] - 2026-04-17

### Added

- **版本重構**：整合影像保險機制與指令優化。
- **核心穩定性提升**：針對特定伺服器環境優化 Shift 移動邏輯。

## [1.2.0] - 2026-04-17

### Added

- **智慧型保險機制**：新增基於 `pyautogui.pixel` 的顏色偵測，自動防範介面卡住或抓取失敗的死循環。

## [1.1.0] - 2026-04-17

### Changed

- **存入邏輯優化**：改用 `/moneysave` 指令取代銀行 GUI 介面點擊。

## [1.0.0] - 2026-04-17

### Added

- **專案初始版本發佈**。
