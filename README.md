# Minecraft Emerald Depositor

*這是 Minecraft 廢土伺服器(mcfallout.net) 中將綠寶石存入銀行的自動化腳本，須搭配紅石拆包機使用。*

## 核心功能

- **優化啟動流程**：初始等待時間移至循環外，啟動後有 5 秒時間切換遊戲視窗，之後即進入循環。
- **智慧保險機制**：
    - 透過監測特定座標（預設 `(958, 226)`）的像素顏色變動，判斷綠寶石是否成功存入。
    - 若偵測到顏色無變動（代表沒存入綠寶石、卡在背包介面），會自動按下 `e` 鍵關閉介面，避免死循環。
- **安全中斷機制**：
    - 隨時按下 `Esc` 鍵立即終止腳本並釋放按鍵。
    - 支援 `FAILSAFE`：將滑鼠移至螢幕左上角即可強制停止。

## 安裝

1. 安裝 Python 3.9+
2. 安裝依賴（v1.2.0+ 需要 Pillow 進行影像處理）：

```bash
pip install -r requirements.txt
```

## 設定與座標 (main.py)

在使用前，請先根據您的螢幕解析度調整 `main.py` 內的座標：

- `COORD_A`：箱子或拆包後的開盒位置。
- `COORD_B`：綠寶石在背包中的位置。
- `target_x, target_y`：用於保險機制監控的像素座標點。

## 啟動

```bash
python main.py
```

## 注意事項與安全性

- **解析度相依**：本工具依賴絕對座標與像素顏色，若解析度改變，需重新設定。
- **保險機制設定**：若預設監控點無效，請自行修改 `target_x` 與 `target_y`。
- **緊急停止**：若腳本失控，請立即按下 `Esc` 鍵或將滑鼠甩至螢幕左上角。
