# Minecraft Emerald Depositor

*這是 Minecraft 廢土伺服器(mcfallout.net) 中將綠寶石存入銀行的自動化腳本，須搭配紅石拆包機使用。*

## 核心功能

- **極速指令輸入**：使用 `pyperclip` 剪貼簿，將 `/moneysave` 透過 `Ctrl+V` 貼上，大幅減少指令輸入時間。
- **智慧保險機制**：
    - 透過監測特定座標的像素顏色變動，判斷綠寶石是否成功移動。
    - 若偵測到顏色無變動，會自動按下 `e` 鍵重置介面，避免死循環。
- **安全中斷機制**：
    - 隨時按下 `Esc` 鍵立即終止腳本。
    - 支援 `FAILSAFE`：將滑鼠移至螢幕左上角即可強制停止。

## 安裝

1. 安裝 Python 3.9+
2. 安裝依賴（v2.1.0+ 新增 `pyperclip`）：

```bash
pip install -r requirements.txt
```

## 設定與座標 (main.py)

在使用前，請先根據您的螢幕解析度調整 `main.py` 內的座標：

- `COORD_A`：界伏盒的開盒位置。
- `COORD_B`：綠寶石在背包中的位置。
- `target_x, target_y`：保險機制監控像素點。

## 啟動

```bash
python main.py
```

## 注意事項與安全性

- **剪貼簿占用**：執行期間會使用剪貼簿複製 `moneysave` 字串。
- **解析度相依**：本工具依賴絕對座標與像素顏色，若解析度改變，需重新設定。
- **緊急停止**：若腳本失控，請立即按下 `Esc` 鍵或將滑鼠甩至螢幕左上角。
