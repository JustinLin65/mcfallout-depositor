# Minecraft Emerald Depositor

*這是 Minecraft 廢土伺服器(mcfallout.net) 中將綠寶石存入銀行的自動化腳本，須搭配紅石拆包機使用。*

## 核心功能

- **自動化循環**：精確模擬「開啟 -> 移動物品 -> 指令存入」的完整流程。
- **極致穩定點擊**：針對 Shift 組合鍵與滑鼠動作進行延遲優化，確保在伺服器延遲時仍能穩定執行。
- **安全中斷機制**：
    - 隨時按下 `Esc` 鍵立即終止腳本並釋放按鍵。
    - 支援 `FAILSAFE`：將滑鼠移至螢幕左上角即可強制停止。
- **視覺化日誌**：終端機即時輸出當前循環次數與執行步驟。

## 安裝

1. 安裝 Python 3.9+
2. 安裝依賴：

```bash
pip install -r requirements.txt
```

## 設定與座標 (main.py)

在使用前，請先根據您的螢幕解析度調整 `main.py` 內的座標：

- `COORD_A`：界伏盒的開盒位置。
- `COORD_B`：綠寶石在背包中的位置（用於 Shift 點擊）。
- `COORD_C`：銀行介面中「存入所有物品」的按鈕位置。

## 啟動

```bash
python main.py
```

## 注意事項與安全性

- **解析度相依**：本工具依賴絕對座標，若遊戲視窗縮放或螢幕解析度改變，需重新設定座標。
- **模組依賴**：需搭配 Inventory Profiles Next 才能實現腳本中一鍵移動所有物品的功能。
- **防範偵測**：請依照個人需求調整 `pyautogui.PAUSE` 與 `safe_sleep` 時間，以模擬更真實的人類操作。
- **緊急停止**：若腳本失控，請立即按下 `Esc` 鍵或將滑鼠甩至螢幕左上角。
